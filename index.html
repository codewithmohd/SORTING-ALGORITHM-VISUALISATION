Algorithms Theory

1. Bubble Sort

Bubble sort compares adjacent elements and swaps them if needed. After each full pass, the largest element moves to its correct position.

2. Insertion Sort

Insertion sort picks elements one-by-one and puts them into the correct position in the sorted portion of the array.

3. Selection Sort

Selection sort finds the minimum element from the unsorted part and places it at the beginning.
